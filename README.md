## ğŸ¯ Customization

### To change colors:
Edit the CSS variables in `styles.css`:
```css
:root {
    --color-primary: #FFE5E5;
    --color-secondary: #FFF8F0;
    --color-accent: #D4AF37;
    --color-lavender: #E6E6FA;
}
```

### To update pricing:
Edit the pricing cards in `index.html` under the `#services` section.

### To change contact info:
Update the contact section in `index.html` and the QR code generation in `script.js`.
